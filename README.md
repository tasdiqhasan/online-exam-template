# 🧪 Online Exam Platform
This project is an online exam platform currently in the early development stage. At the moment, it focuses on building the HTML template for the user interface.

## 📌 Current Progress
Static HTML layout for:
- [x] Login page
- [x] Welcome Screen
- [ ] Exam page
- [ ] Result page
- [ ] Admin Dashboard
- [ ] User Management
- [ ] Question Bank

## 🛠️ Tech Stack
- HTML
- Tailwind CSS

## ▶️ How to Run
Simply open the `login.html` file in your browser.
